# Ejemplos Básicos de Programación en C++ (Nivel Estructurado)

Este repositorio contiene 5 ejemplos básicos de programación en C++ que ilustran conceptos estructurados como entrada/salida, bucles y condicionales.

## Ejemplos

1. **ejemplo1.cpp**: Hola Mundo - Imprime un mensaje simple.
2. **ejemplo2.cpp**: Suma de dos números - Lee dos números y calcula su suma.
3. **ejemplo3.cpp**: Bucle for - Imprime números del 1 al 10.
4. **ejemplo4.cpp**: If-else - Verifica si un número es par o impar.
5. **ejemplo5.cpp**: Bucle while - Cuenta del 1 al 5.

## Compilación y Ejecución

Para compilar y ejecutar cada ejemplo, usa los siguientes comandos en una terminal:

```bash
g++ ejemplo1.cpp -o ejemplo1
./ejemplo1
```

Reemplaza `ejemplo1` con el número correspondiente.

## Requisitos

- Compilador C++ (como g++).
- Sistema operativo Linux.

## Troubleshooting

- Si obtienes errores de compilación, asegúrate de que g++ esté instalado: `sudo apt install g++`.
- Para ejemplos con entrada (ejemplo2 y ejemplo4), proporciona input cuando se ejecute.
